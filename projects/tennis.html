Tennis Surface study

Developed as a part of final project for Information Visualization class in a team of three with Puneet Sharma and Proxima DasMohapatra.   Code   Website   Full Report

Problem:

One of the most interesting aspects of the game of tennis is the court surface itself and how different players adopt different playing styles in order to increase their chances of winning. So for this project we decided to present a study of these three main surfaces, Grass (at Wimbledon), Clay (at the French Open) and Hard courts (US Open and Australian Open) and various metrics that highlight a certain aspect of the game.

Process:

The first step in the process was studying previous work and research done in the field of sports info graphic and information visualization. We specifically looked into tennis visualization and the affects of court surfaces on playing style. (Please see the full report for detailed analysis of related work)

After deep dive into related work and research we had to look for a dataset that would serve the purpose of explaining surface. We found a lot of data sets containing tennis stat. We finally used a point-by-point game data of 5 years of grand slams (Data source). The data had to be cleaned and compiled in a form that could be used for visualizing. We used different tools for the purpose, starting from simple manipulation in spreadsheet to Python, R and MySql.

One of the most important aspect of information visualization that captures audience attention is creating a narrative around the information. We recognized the need to explain the game and giving a background of the tournament in order to set a stage for the presentation. To accomplish this purpose we decided to start the presentation with info graphic. This info graphic was first designed on paper in several lo-fi prototypes. We finalized the prototype using adobe illustrator after getting feedback and revising the paper prototype. The info graphic gave introduction to the tournament, different surfaces and the effect of surface on game.

For next part of the presentation we wanted to orient the audience geographically. For this we created an interactive map in D3 highlighting all the place where tennis tournaments in different court (hard, grass or clay) happen around the world. It also highlighted the relative abundance of hard court compared to grass and clay court.

After the map we wanted to show the four metrics to highlight and the affect of surface on the game. For this purpose we created four interactive dashboards and storyboards using Tableau. Users could interact and explore interesting patterns in the dashboard. We also added supporting text and explanation with the dashboard.

For the last part of the visualization, we brought together all the data in an interactive parallel coordinate chart to show how these metrics relate to each other.

Skills used:

  • Design in Illustrator
  • Data Cleaning
  • Data Analysis
  • Data Visualization - Highcharts, D3, Tableau
  • Frontend Development - HTML, CSS, Javascript
